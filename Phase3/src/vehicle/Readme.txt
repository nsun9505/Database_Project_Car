2. 차량 관련 기능
 A. 회원은 로그인 이후 차량 매물 전체를 볼 수 있다.
   > 차량 매물 출력 시 모든 정보를 출력할 필요가 없다.
   > 그러므로 DB에서 필요한 COLUMN 정보만 가져온다.
     Ex. [등록번호] [제조사] [세부 모델] [FUEL] [MODEL_YEAR] [MILEAGE] [LOCATION] [가격]
   > 필요한 정보를 출력하기 위한 DTO 생성 : BasicVehicleInfo
	Class BaiscVehicleInfo
		registration_number
		make
		detailed_model_name
		model_year
		mileage
		location
		fuel
   > 1. 매물 검색 	2. 차량 정보 보기	3. 다음 페이지  4. 이전 페이지  5. 로그아웃  6. 회원 정보 변경
   > 선택 : 
 B. 회원은 로그인 이후 제조사별로 등록된 차량을 검색하는 것이 가능하다.
 C. 회원은 로그인 이후 특정 조건으로 차량을 검색할 수 있다.
 -> 합쳐서 생각
  > 검색의 조건을 제조사로 하여 검색
  > 또는 다른 추가 조건을 조합하여 매물을 검색한다.
  > 1. 매물 검색  2. 차량 정보 보기 3. 다음 페이지  4. 이전 페이지 5. 로그아웃 6. 회원 정보 변경
  > 선택 : 1번 선택!
  > 조건 검색
  > 1.차종 2.제조사 3. 모델 4. 세부 모델 5. 
  > 제조사 : 1. 기아 2. 르노삼성 3. 현대 4. 쉐보레 5. 아우디 6. 벤츠 7. 폭스바겐 8. BMW 9. 건너뛰기
  > 제조사 선택 : 
  > 
 D. 검색한 차량 목록에서 차량을 선택하면 차량의 정보(모든 정보)를 확인할 수 있으며, 구매 가능
  > 1. 매물 검색  2. 차량 정보 보기 3. 다음 페이지  4. 이전 페이지 5. 로그아웃 6. 회원 정보 변경
  > 선택 : 2번 선택
  > 등록번호 : 
  > 등록번호에 맞는 정보를 데이터베이스에서 찾아서 출력하고 구매하기, 뒤로가기 기능 추가

 E. 회원이 구매한 차량은 이후 검색 대상에서 제외한다.
  > 매물 검색할 때마다 ORDER_LIST에 있는 것은 제외하므로 자동으로 해결

 F. 회원은 차량 이름 검색으로 특정 차량을 검색하는 것이 가능하다.
  > 차량 이름은 model_name 또는 detailed_model_name으로 검색가능
  > 매물 검색에서 3. 모델 또는 4. 세부 모델을 선택하여 나오는 항목에서 검색할 수 있다.
